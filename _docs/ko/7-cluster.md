클러스터의 상태와 리소스, 서비스, 알람이 제공된다.
![]({{ site.baseurl }}/assets/KR/{{ site.version }}/7_1.png)

| 클러스터 메뉴 | 설명                                                        |
| :------ | :-------------------------------------------------------- |
| 리소스      | 노드풀, 스토리지, 게이트웨이의 정보 서비스         |
| 서비스  | 전체 서비스의 CPU, Memory, Network 사용량 및 개별 서비스의 모니터링 서비스 |
| 알람      | 클러스터의 이벤트 정보 관련 알람 서비스                                    |
